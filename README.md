# What is this ?

Get the user's information at anytime you want.


# Installation

`npm i fingerprintjs --save`

Then...

```
import {fingerprint_browser} from 'fingerprintjs';

    fingerprint_browser(browser);

```


## Options

fingerprint js support one option which is mandatory to get the result on the screen.

* *Element ID* enter the element id so that the result can be shown on the main screen
